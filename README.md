### Pasos para usar

Requisitos:

1. Visual Studio en Windows
2. Docker

Pasos:

1. Abrir el proyecto después de clonarse
2. Ejecutar la base de datos PostgreSQL usando docker

```bash
docker compose up -d
```

3. Ejecutar la aplicación con ayuda de Visual Studio ![image](https://learn.microsoft.com/en-us/visualstudio/ide/media/visualstudio/csharp-winform-visual-studio-start-run-program.png?view=visualstudio)
